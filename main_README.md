# 🩺 Medical AI Practice
의공학 전공자가 **의료 영상 AI 개발**을 이해하기 위해 진행한 실습 프로젝트 모음입니다.  
의료 영상 데이터를 실제로 다뤄보고, 간단한 AI 모델을 구현하는 과정을 통해  
의료 AI 파이프라인의 기초를 학습하는 것을 목표로 합니다.

---

## 📂 프로젝트 구성

| No. | 제목 | 내용 | 기술 포인트 |
|-----|------|------|--------------|
| 1️⃣ | X-ray CNN Classification | 흉부 X-ray로 정상/폐렴 분류 | PyTorch, CNN, DataLoader |

- matplotlib / numpy
